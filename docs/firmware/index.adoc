== Overview

BeeLight is a battery-powered Zigbee environmental sensor based on the Nordic Semiconductor nRF54L15 SoC.
The device measures illuminance, temperature, air pressure, humidity, and battery voltage, and transmits all readings over a Zigbee network to a coordinator (e.g. Home Assistant via Zigbee2MQTT).

image::Rendering.png[BeeLight 3D Rendering, 600, align="center"]

=== Features

* *MCU*: Nordic Semiconductor nRF54L15 (ARM Cortex-M33, 128 MHz)
* *RTOS*: Zephyr RTOS with nRF Connect SDK 2.9.2
* *Protocol*: Zigbee 3.0 – Home Automation Profile (ZBOSS Stack via ncs-zigbee)
* *Sensors*:
** Broadcom APDS-9306 – Illuminance measurement
** Bosch BME688 – Temperature, air pressure, humidity
** Bosch BME688 (BSEC2, optional) – CO₂ equivalent, VOC, IAQ
* *Power supply*: CR2 battery (~3.3 V), voltage monitoring via ADC
* *Firmware version*: 2.0.0

== Integration

=== Home Assistant

The device registers itself automatically with Home Assistant as a sensor device after joining the Zigbee network.
All measured values are available as entities.

image::HomeAssistant.png[Home Assistant Integration, 600, align="center"]

=== Zigbee2MQTT
